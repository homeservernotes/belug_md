# belug markdown

This repository contains source markdown and css files for the belug website and the support files/scripts needed to generate html using either lowdown or jekyll.

## html generation using lowdown:

Install a version of lowdown that supports templates.  Template support may require building from source. Run `tools/make_www' to create the html file in directory _lowdown. Run'tools/view_www` to view them.  A version of lowdown that supports templates is required.  

## html generation using jekyll:

Install jekyll & bundle. Run "bundle exec jekyll build" to create the html files in directory "_site" .   Run "bundle exec jekyll serve -H 0.0.0.0" to serve the site on port 4000 which can be viewed from the browser.
